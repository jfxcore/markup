# JFXcore.markup
Runtime support library for FXML 2.0 markup. This library is only required for FXML 2.0 documents that use
[MarkupExtension](https://jfxcore.github.io/fxml-compiler/markup-extension.html) or the `InverseMethod`
annotation for compiled bindings.

# Releases
The latest release is available on [Maven Central](https://central.sonatype.com/artifact/org.jfxcore/markup/0.1.0).

## Maven
```xml
<dependency>
    <groupId>org.jfxcore</groupId>
    <artifactId>markup</artifactId>
    <version>0.1.0</version>
</dependency>
```

## Gradle
```kotlin
dependencies {
    implementation("org.jfxcore:markup:0.1.0")
}
```
